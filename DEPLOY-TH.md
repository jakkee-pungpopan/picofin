# วิธีเอา ADDBANK ขึ้นออนไลน์ให้คนอื่นใช้ (ฉบับจับมือทำ)

> ระบบนี้เป็นตัวอย่าง เงินทั้งหมดเป็นเงินจำลอง (sandbox) ไม่ใช่ธุรกรรมจริง

ภาพรวม: เราจะเอา **ระบบหลังบ้าน + ฐานข้อมูล** ขึ้นเว็บ Render (ฟรี) แล้วเอา **เว็บแอป** ขึ้น Netlify (ฟรี)
สิ่งที่คุณต้องสมัครเอง (ผมสมัครแทนไม่ได้): GitHub, Render, Netlify — ทั้งหมดสมัครฟรี ใช้บัญชี Google ล็อกอินได้

---

## สิ่งที่ต้องมี
- บัญชี GitHub (github.com) — ฟรี
- บัญชี Render (render.com) — ฟรี ล็อกอินด้วย GitHub ได้เลย
- บัญชี Netlify (netlify.com) — ฟรี

---

## ขั้นที่ 1 — อัปโค้ดขึ้น GitHub
1. ไปที่ github.com → New repository → ตั้งชื่อ `addbank` → Create
2. กดปุ่ม **"uploading an existing file"** บนหน้า repo
3. ลากโฟลเดอร์ `novabank` ทั้งโฟลเดอร์ (ที่แตกจาก zip) เข้าไปวาง → Commit

## ขั้นที่ 2 — เปิดระบบหลังบ้าน + ฐานข้อมูล บน Render
1. ไปที่ render.com → ล็อกอินด้วย GitHub
2. กด **New → Blueprint**
3. เลือก repo `addbank` → Render จะอ่านไฟล์ `render.yaml` ให้เอง
   (มันจะสร้างให้ 2 อย่าง: ฐานข้อมูล `addbank-db` + เซิร์ฟเวอร์ `addbank-api`)
4. กด **Apply** แล้วรอ build เสร็จ (~5-10 นาทีครั้งแรก)
5. เสร็จแล้วคลิกที่บริการ `addbank-api` → คัดลอก URL ด้านบน เช่น
   `https://addbank-api.onrender.com`
6. ทดสอบ: เปิด `https://addbank-api.onrender.com/api/docs` ในเบราว์เซอร์ → ต้องเห็นหน้า Swagger

> หมายเหตุ Render ฟรี: เซิร์ฟเวอร์จะ "หลับ" เมื่อไม่มีคนใช้ ครั้งแรกที่เปิดอาจช้า ~50 วินาที ถือว่าปกติ

## ขั้นที่ 3 — เอาเว็บแอปขึ้น Netlify
1. ไปที่ app.netlify.com/drop
2. ลากไฟล์ `novabank/apps/web-app/index.html` (หรือทั้งโฟลเดอร์ `web-app`) เข้าไปวาง
3. Netlify จะให้ลิงก์เว็บมา เช่น `https://addbank-xxxx.netlify.app` — **นี่คือลิงก์ที่ส่งให้คนอื่นใช้**

## ขั้นที่ 4 — เชื่อมเว็บแอปกับเซิร์ฟเวอร์
1. เปิดลิงก์เว็บแอปที่ได้จาก Netlify
2. หน้าแรกจะให้ใส่ "ที่อยู่เซิร์ฟเวอร์" → วาง URL จากขั้นที่ 2 (เช่น `https://addbank-api.onrender.com`) → บันทึก
3. ล็อกอินด้วยบัญชีตัวอย่าง:  `somchai@addbank.local` / `Password@123`
   หรือกด "สมัครสมาชิกใหม่" เพื่อสร้างผู้ใช้ของคุณเอง
4. ลองโอนเงิน → ยอดเงินจะถูกบันทึกถาวรในฐานข้อมูลจริง ✅

---

## เสร็จแล้วได้อะไร
- เว็บแอป ADDBANK ที่เปิดจากมือถือ/คอมเครื่องไหนก็ได้ผ่านลิงก์
- สมัคร/ล็อกอิน/โอนเงิน บันทึกข้อมูลถาวร รองรับหลายคน
- หน้า Admin ดูข้อมูลรวมได้ (deploy เพิ่มได้ภายหลังด้วยวิธีเดียวกัน)

## ค่าใช้จ่าย
- เริ่มต้นฟรีทั้งหมด (Render + Netlify free tier)
- ฐานข้อมูลฟรีของ Render มีอายุ ~90 วัน (เหมาะกับการทดลอง/เดโม) ถ้าจะใช้ยาวค่อยอัปเกรดเป็นแบบเสียเงินภายหลัง

## ข้อจำกัด (พูดตรงๆ)
- นี่คือระบบจำลอง ไม่เชื่อมเงินจริง/ธนาคารจริง/พร้อมเพย์จริง
- การทำให้โอนเงินจริงได้ ต้องมีใบอนุญาตธนาคารและอยู่ภายใต้การกำกับของ ธปท. ซึ่งเกินขอบเขตของโค้ด
